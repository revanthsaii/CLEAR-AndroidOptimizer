> [1.0.0] `INITIAL RELEASE`
>
> - Android system optimization module for Magisk, KernelSU, APatch, and variants
> - Dalvik cache cleanup for improved app performance
> - System dropbox log removal
> - Temporary file deletion across cache directories
> - Tombstone log cleanup
> - Empty directory cleanup in Android folder
> - Thumbnail cache clearing
> - System and shader cache cleaning
> - FSTRIM partition optimization for enhanced I/O performance
> - RAM cache clearing using drop_caches
> - All-in-one optimization option
> - Supports Android 10+ (SDK 29+)
> - Command-line interface via Termux: `su -c CLEAR`
